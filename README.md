# Desh Tours - Udawalawa Safari Website

## Features
- 🐘 Modern safari travel agency website
- ⭐ Customer review system with local storage
- 📸 Photo upload for feedback submissions
- 🖼️ Dynamic gallery from user photos
- 🌙 Light/Dark theme toggle
- 📍 Google Maps integration

## Technologies Used
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Font Awesome Icons
- AOS Scroll Animations

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. All data saved in localStorage
